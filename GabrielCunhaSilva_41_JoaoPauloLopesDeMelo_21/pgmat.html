Matérias do Curso

Disciplinas trabalhadas no Técnico em Desenvolvimento de Sistemas

Técnicas de Programação e Algoritmo

Introduz os alunos à lógica de programação, algoritmos, estruturas de controle e resolução de problemas computacionais.

Banco de Dados

Envolve modelagem, criação e manipulação de bancos de dados relacionais utilizando SQL e ferramentas como SQL Server.

Análise e Projeto de Sistemas

Trabalha com modelagem de sistemas utilizando diagramas UML, levantamento de requisitos e documentação técnica.

Design Digital

Introdução ao design gráfico para criação de interfaces e elementos visuais, com uso de ferramentas como Photoshop e Canva.

Fundamentos da Informática

Ensina o funcionamento básico do computador, sistemas operacionais, software e hardware, e uso geral da tecnologia.

Programação Web

Aborda o desenvolvimento de sites e aplicações usando HTML, CSS, JavaScript, PHP, frameworks e boas práticas de desenvolvimento.

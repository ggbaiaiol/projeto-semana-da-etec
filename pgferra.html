Ferramentas

Utilizadas no Curso Técnico de Desenvolvimento de Sistemas

SQL Server Management Studio 20

Ambiente completo para gerenciar bancos de dados SQL Server, criar consultas, tabelas e procedimentos armazenados.

brModelo

Ferramenta utilizada para modelagem de banco de dados com diagramas Entidade-Relacionamento (MER).

Visual Studio Code

Editor de código leve e versátil usado para desenvolvimento web, com suporte para múltiplas linguagens.

Visual Studio 2022

IDE poderosa para desenvolvimento de aplicações Windows e Web em C#, .NET, ASP.NET, entre outros.

Portugol Studio

Ambiente de desenvolvimento voltado para introdução à lógica de programação, com sintaxe semelhante ao português.

Adobe Photoshop CS6 (64 Bit)

Ferramenta profissional para edição de imagens e criação de materiais gráficos para projetos visuais.

Canva

Ferramenta online de design gráfico, usada para criar apresentações, logos, banners e layouts de forma intuitiva.

Oracle VM VirtualBox

Software de virtualização utilizado para simular ambientes operacionais diversos para testes e aprendizado.

Oracle VM VirtualBox

O Oracle VirtualBox é um software gratuito e de código aberto que permite criar e executar máquinas virtuais (VMs) com diferentes sistemas operacionais (como Windows, Linux, macOS e Android) num único computador físico (host). É uma ferramenta popular para desenvolvimento, testes e suporte de aplicações, oferecendo uma interface gráfica fácil de usar e integração com a Oracle Cloud Infrastructure (OCI).


Principais características:

Virtualização multiplataforma:
Permite executar vários sistemas operacionais simultaneamente ou individualmente num sistema hospedeiro (Windows, macOS ou Linux).

Código aberto e gratuito:
O VirtualBox é um projeto de código aberto, o que o torna amplamente acessível para uso pessoal e comercial.

Interface gráfica e linha de comando:
Possui uma interface de utilizador gráfica (GUI) intuitiva e uma interface de linha de comando (CLI) para facilitar o gerenciamento e a automação.

Suporte para nuvem:
Integra-se com a Oracle Cloud Infrastructure (OCI), permitindo o upload e download de VMs na nuvem com um único clique.

Suporte a hardware:
Suporta drivers para hardware específico, como USB, e oferece melhorias de 3D para máquinas virtuais.

Criação de ambientes de teste:
É ideal para equipas de desenvolvimento testarem aplicações em diferentes sistemas operativos sem a necessidade de múltiplos dispositivos físicos.

Como funciona:
O VirtualBox simula um computador completo com disco rígido, memória, processador e outros componentes. O sistema operacional convidado (a VM) é instalado e executado dentro deste ambiente simulado, utilizando os recursos do hardware real do computador hospedeiro. Isso permite que o utilizador tenha acesso a vários sistemas operacionais numa única máquina.
